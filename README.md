Penrose Pages is a journaling platform that integrates technology and psychiatry. It is designed to help users track emotional patterns, reflect deeply, and gain insights into their mental health through personalized recommendations and clinical self-assessment tests.

---

## Key Features

- **Journal Management:** Create, edit, delete, and paginate journal entries.  
- **Emotional Tracking via Sentiment Analysis:** Detect and track emotional patterns in users' journal entries.  
- **Personalized Recommendations:** Suggest books, meditations, and quotes based on the frequency of emotions in entries.  
- **Mental Health Self-Assessments:** Includes psychiatry-backed tests like,
  - **GAD-7** for Anxiety
  - **PHQ-9** for Depression
  - **CBI** for Burnout
- **Visual Analytics:** Chart test scores over time to monitor progress or decline in mental health.  

---

## Tech Stack

- **Backend:** Node.js | Express | MongoDB | REST APIs | sessions  
- **Frontend:** HTML | CSS | JavaScript  
- **JavaScript Libraries:** Sentiment.js for NLP | Chart.js for Visualization
