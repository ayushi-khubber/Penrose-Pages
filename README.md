Penrose Pages is a journaling platform that integrates tech and psychiatry. It's designed to help users track emotional patterns, introspect deeply, and gain insights into their mental health through personalized recommendations and clinical self-assessment tests.

---

## Key Features

-  **Journal Management:** Create, edit, delete, and paginate journal entries
-  **Emotional Tracking via Sentiment Analysis:** Detect and track emotional patterns in users' journal entries
-  **Personalized Recommendations:** Suggest books, meditations, and quotes based on the frequency of emotions in the journal entries  
-  **Mental Health Self-Assessments:** Self-assess mental health with psychiatry-backed clinical tests like, **GAD-7** for Anxiety, **PHQ-9** for Depression, and **CBI** for Burnout
- **Visual Analytics:** Visualize the test results by plotting the scores received over time, helping users keep track of any progress or decline in their mental health  

---

## Tech Stack

- **Backend:** Node.js | Express | MongoDB | REST APIs | sessions  
- **Frontend:** HTML | CSS | JavaScript  
- **JavaScript Libraries:** Sentiment.js for NLP | Chart.js for Visualization
